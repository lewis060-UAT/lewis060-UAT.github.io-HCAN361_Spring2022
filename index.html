Using the Healthcare Cost and Utilization Project's (HCUP) Nationwide Inpatient Survey (NIS)

This website highlights findings from students enrolled in HCAN 361 during the Spring 2022 Academic Semester. For this year's project, Juniors in The University of Alabama's Health Care Analytics Concentration used the 2019 Healthcare Cost and Utilization Project's (HCUP) Nationwide Inpatient Survey (NIS) to examine national trends for various health outcomes. The final deliverable in this project is an infographic.

What do we do in this course

learn about data

We learn "why" datasets have various structures, before learning "how" to modify and query data.
‍

work with databases

Most (if not all) healthcare organizations rely on secured servers to store sensitive and proprietary data. We learn the basics of how to use these kinds of systems.

manage data projects

Learning how to code is not enough in today's world of "big data." We learn the project management aspect of successfully facilitating an analytics project.

In a 4-week final exam "mini" capstone project,  students  applied skills learned in HCAN 361 to query data in a Microsoft SQL Server using Azure Data Studio as the primary data management platform. Data sources, which were loaded onto a server, were accessed through a secured remote desktop setup comparable to the computing environments of most health care organizations. The full details of used data sources are detailed in a notebook accessible by clicking the button below.
